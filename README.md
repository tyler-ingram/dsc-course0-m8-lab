# Aviation Accident Analysis

### Structure
The data folder holds the data being analyzed from 2023 aviation accidents. There is a notebook where the data is cleaned and a notebook doing the visualizations and analysis. 

### Summary of findings
From the data, it can be seen that in general larger airplanes (More than 20 passengers) are safer than smaller airplanes when you get into an accident. This can be seen both in the percent of serious injuries and fatalities and also the percent of planes destroyed in accidents.
